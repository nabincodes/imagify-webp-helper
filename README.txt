== Description ==

This is the helper plugin that automatically avoid the conversion of <img> to <picture> tag when Imagify's WebP feature is enabled.

== Installation ==

1. Upload `imagify-webp-helper.zip`  through the Plugins menu in WordPress
1. Activate the plugin after installation

== Changelog ==
